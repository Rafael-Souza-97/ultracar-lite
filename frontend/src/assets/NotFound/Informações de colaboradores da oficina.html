Aqui está uma possível solução para uma máscara de data no formato "xx/xx/xxxx":

javascript
function handleDate(event) {
  const inputDate = event.target.value;
  const formattedDate = inputDate
    .replace(/\D/g, '') // remove todos os caracteres não numéricos
    .replace(/(\d{2})(\d)/, '$1/$2') // adiciona uma barra após os dois primeiros números
    .replace(/(\d{2})(\d)/, '$1/$2') // adiciona uma barra após os quatro primeiros números
    .slice(0, 10); // limita a string a 10 caracteres (xx/xx/xxxx)

  setOrder((prevState) => ({ ...prevState, date: formattedDate }));
}

Note que esta função remove todos os caracteres não numéricos do input, adiciona as barras em suas posições apropriadas e limita o tamanho da string a 10 caracteres (correspondente ao formato xx/xx/xxxx). Além disso, a função atualiza o estado de uma ordem de serviço com a data formatada. Você pode adaptá-la para atender às suas necessidades específicas.

Em seguida, você pode usar o método filter do array para filtrar todos os objetos do mock que contêm o nome do mecânico. Aqui está um exemplo de como você pode fazer isso:

javascript
import { useParams } from 'react-router-dom';

function TabelaMecanico() {
  const { nome } = useParams();

  const mock = [
    {
      name: 'João da Silva',
      cpf: '123.456.789-00',
      cel: '(11) 99999-9999',
      office: 'Mecânico',
      wage: '3.000,00',
      startDate: '01/01/2020',
      benefits: 'Vale transporte',
      address: 'Rua A, nº 123'
    },
    // ...
  ];

  const filteredMock = mock.filter((mecanico) =>
    mecanico.name.toLowerCase().includes(nome.toLowerCase())
  );

  return (
    <div>
      <h2>Mecânicos com nome {nome}</h2>
      <table>
        <thead>
          <tr>
            <th>Nome</th>
            <th>CPF</th>
            <th>Celular</th>
            <th>Cargo</th>
            <th>Salário</th>
            <th>Data de Início</th>
            <th>Benefícios</th>
            <th>Endereço</th>
          </tr>
        </thead>
        <tbody>
          {filteredMock.map((mecanico) => (
            <tr key={mecanico.cpf}>
              <td>{mecanico.name}</td>
              <td>{mecanico.cpf}</td>
              <td>{mecanico.cel}</td>
              <td>{mecanico.office}</td>
              <td>{mecanico.wage}</td>
              <td>{mecanico.startDate}</td>
              <td>{mecanico.benefits}</td>
              <td>{mecanico.address}</td>
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
}

Observe que a função includes é usada para verificar se o nome do mecânico contém a sequência de caracteres especificada na URL, ignorando maiúsculas e minúsculas.
